시스템 점검 시 자동화 진단을 위한 코드
보안포탈에 자동으로 데이터베이스 등록 및 필요 항목에 대해서만 진단이 가능하며
개발 최종 목표는 원격으로 보안포탈을 통해서 진단하는것이 목표임

[목표]
1. MySQL Code 작성 예정
2. 세부 항목 진단을 위한 분류 예정 (높은 용량과 Line에 따른 에러 방지)
3. 중복 항목(예:snmp점검)에 대한 리스트 블록으로 묶기
3. 보안포탈 내 현황 업로드 예정

[진행사항]
- 2024년 전자금융기반시설 Linux 기반으로 작성함
- 항목에대한 전체 코드 작성 완료
- def으로 세부 분류 완료
- 결과파일 테스트중
